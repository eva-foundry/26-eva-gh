# Accenture Jurisprudence MVP – Feature List (Extract)

## Web Application
- User authentication
- Chat-based query interface
- Display answers with citations
- Case reference links to source documents
- Chat history per session

## Backend / API
- Question routing to RAG backend
- Prompt templates for jurisprudence queries
- Citation enforcement in responses

## Data & Ingestion
- Incremental refresh using CanLII metadata (CDC)
- Support for PDF and HTML case documents
- Deterministic re-runs of ingestion

## Explicitly Out of Scope (for MVP)
- Multilingual UI
- Accessibility enhancements
- Advanced analytics dashboards
