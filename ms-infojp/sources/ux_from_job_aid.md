# Current Jurisprudence UX – Job Aid (Extract)

## Core UX Patterns
- User signs in
- User submits a jurisprudence question
- System shows processing state
- Answer displayed in chat format
- Citations shown inline or below answer
- Links to source case documents
- Chat history retained for the session
- Optional user feedback on answer usefulness

## Observations
- UX is functional but verbose
- Navigation between results and sources is clunky
- Opportunity for clearer citation presentation
